## Text Formatting

Regular text looks like this. Lorem ipsum dolor sit amet, consectetur adipiscing elit.

**Bold text** looks like this. *Italic text* looks like this.

~~Strikethrough text~~ uses tildes.

`Inline code` uses backticks.

- [ ] Incomplete task
